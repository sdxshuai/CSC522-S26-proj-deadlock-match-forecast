# Midway Report Grading Criteria

## Problem Statement (5 points)

Clearly describes the problem statement and motivates the need for the proposed analysis on the given dataset.

## Related Work (5 points)

At least 3 relevant scholarly works are referenced, explained and connected to the current problem and approach.

## Approach (15 points)

Clearly explains the machine learning techniques that were applied to this problems with specific details, and highlights the novelty of the approach.

## Rationale (10 points)

Clearly explains the machine learning techniques that were applied to this problems with specific details, and highlights the novelty of the approach.

## Dataset(s) (5 points)

Clearly explains the machine learning techniques that were applied to this problems with specific details, and highlights the novelty of the approach.

## Hypotheses (5 points)

Describes 1 or more hypotheses that are interesting, non-obvious, relevant to the problem description, and which can be answered with the proposed approach.

## Experimental Design (10 points)

Clearly explains the procedures/experiments which were implemented to evaluate these hypotheses, incluing reproducable steps (and preprocessing). This should include a description of any software artifacts created.

## Partial Results (10 points)

Results from the completed experiments are presented with appropriate tables and visuals.

## Design of Future Experiments (20 points)

Clearly explains the procedures/experiments which will be implemented to evaluate the hypotheses. This should include a description of any software artifacts created.

## Hyperparameters Appendix (10 points)

Tuning Approach Appendix included with all 5 required elements for each classifier. Rationale is included as well.

## Plan of Activities (5 points)

Gives a step-by-step account of activities necessary to implement the future experiments, with roles for each team member.

## Report Length

Page limit not exceeded (5 pages).

## References

References correctly formatted

## Paper Format

Paper in ACM Format
